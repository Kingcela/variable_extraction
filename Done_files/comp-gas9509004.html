Abstract

In order to investigate the evolutionary process of many deterministic dynamical systems with unfixed parameter, a set of dynamical models with parameter changing continuously and the accumulation of this change might be large is introduced and discussed. The boundary crises and the period-doubling bifurcations are found suppressed and scaling properties for these phenomena are exploited. Due to this suppression, the period-doubling bifurcations seem no longer continuous. We further discuss the possible applications of these models.

Possible applications There are various applications of the developing diagram. Summary and discussion In summary, we have introduced a set of dynamical models with parameter changing continuously and the accumulation of this change might be large. The parameters for boundary crises and period-doubling bifurcations are shifted and scaling properties for these phenomena are exploited. There are still a number of interesting questions remained to be investigated. Since the data series from our model (as well as many natural systems) are the set of transients for different parameters, how to extend the general methods[see, eg. 21,22] to extract information from the data series and then control and predict them is needed.
